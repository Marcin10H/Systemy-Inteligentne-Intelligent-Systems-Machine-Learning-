# Projekt 4: Decision Trees & Rule-based Systems 🌳

## Opis projektu

Projekt bada alternatywne do sieci neuronowych podejście w Machine Learningu, wykorzystując Drzewa Decyzyjne do konstrukcji systemów ekspertowych. Główną zaletą tego podejścia jest wysoka interpretowalność modelu (tzw. white-box model), która pozwala na bezpośrednie wyodrębnienie logicznych reguł decyzyjnych rządzących klasyfikacją danych.

### 🛠️ Technologie i Środowisko
Modele sieci neuronowych oraz proces ich uczenia zostały zrealizowane w dedykowanym środowisku symulacyjnym (pliki `.acid`). 

### 🛠️ Architektura i Metodologia

* **Zbiory danych:** Analizie poddano przestrzenie 2D (liniowo separowalne i nieseparowalne), dane wielowymiarowe (obrazy monochromatyczne) oraz bazy medyczne.
* **Budowa drzew:** Generowanie pełnych drzew decyzyjnych zapewniających 100% dokładności na zbiorze treningowym (ryzyko overfittingu).
* **Ekstrakcja reguł:** Transformacja ścieżek grafu drzewa do postaci jawnych, użytecznych reguł typu `IF-THEN` (np. `JEŻELI x1 < -4.64 TO Klasa 3`).
* **Wizualizacja przestrzeni:** Mapowanie dwuwymiarowej przestrzeni danych w celu nałożenia wygenerowanych przez drzewo obszarów decyzyjnych i porównania ich z obszarami generowanymi przez perceptron wielowarstwowy.

### 🧪 Wyniki

Drzewa decyzyjne pozwoliły na wygenerowanie kompletnej i wysoce czytelnej bazy reguł dla badanych problemów. Model dla danych separowalnych wyodrębnił krystalicznie czyste granice decyzyjne, skutecznie radząc sobie z przypisaniem obiektów. W przypadku bardziej złożonych danych zaobserwowano ortogonalny charakter podziału przestrzeni, specyficzny dla algorytmów opartych na drzewach, co stanowiło interesujący kontrast wizualny względem gładkich granic generowanych przez sieci neuronowe.

## English Description

This project explores Decision Trees as a powerful, interpretable alternative to neural networks in Machine Learning. As a "white-box" model, decision trees allow for the direct extraction of logical rules that govern data classification, providing transparency that deep learning models lack.

### 🛠️ Tools & Environment
The neural network models and their training process were implemented in a dedicated simulation environment (`.acid` files). 

### 🛠️ Architecture & Methodology

* **Datasets:** Analyzed 2D spaces (both linearly separable and inseparable), multi-dimensional image data, and medical databases.
* **Tree Construction:** Generating full decision trees to ensure 100% training accuracy.
* **Rule Extraction:** Translating the tree's graph paths into actionable `IF-THEN` rules (e.g., `IF x1 < -4.64 THEN Class 3`).
* **Space Visualization:** Mapping the 2D data space to visualize the decision boundaries generated by the tree and comparing them with those created by Multi-Layer Perceptrons.

### 🧪 Results

The decision trees generated complete, highly legible rule bases for the classification problems. For separable data, the model established crystal-clear, orthogonal decision boundaries. This geometric division of the feature space provided a stark, fascinating contrast to the smooth, curved boundaries typically generated by neural network activations.
