# Systemy Inteligentne | Intelligent Systems & Machine Learning 🧠

## O projekcie

Repozytorium zawiera zbiór projektów badawczych i implementacji z zakresu **Sztucznej Inteligencji** i **Uczenia Maszynowego**, zrealizowanych w ramach kursu Systemy Inteligentne 2. Skupiają się one na projektowaniu, trenowaniu i testowaniu klasyfikatorów dla danych dwuwymiarowych i wielowymiarowych (rozpoznawanie obrazów).

### 📁 Zawartość repozytorium:

* **Projekt 1: Perceptron Wielowarstwowy (MLP)** – Klasyfikacja danych dwuwymiarowych, liniowo separowalnych przy użyciu sztucznych sieci neuronowych.
* **Projekt 2: Sieci Neuronowe dla danych nieliniowych** – Konstrukcja systemu decyzyjnego dla danych liniowo nieseparowalnych. Analiza aktywności neuronów w warstwach ukrytych.
* **Projekt 3: Rozpoznawanie Obrazów (ANN)** – Klasyfikacja wielowymiarowa obrazów monochromatycznych (rozpoznawanie liter i cyfr). Analiza metryk skuteczności: *Accuracy, Sensitivity (TPR), Specificity (TNR)* oraz Macierzy Pomyłek.
* **Projekt 4: Drzewa Decyzyjne** – System klasyfikujący dane za pomocą pełnych drzew decyzyjnych wraz z wyznaczaniem obszarów decyzyjnych i bazy reguł.
* **Projekt 5: Sieć Hopfielda** – Pamięć autoasocjacyjna. Zastosowanie sieci do odtwarzania i odszumiania zniekształconych obrazów znaków graficznych przy użyciu m.in. reguły Hebba i pseudoinwersji.

## About the project

This repository contains a collection of research projects and implementations in the field of **Artificial Intelligence** and **Machine Learning**. The projects focus on designing, training, and testing classifiers for 2D and multi-dimensional data (including image recognition).

### 📁 Repository content:

* **Project 1: Multilayer Perceptron (MLP)** – Classification of 2D linearly separable data using Artificial Neural Networks.
* **Project 2: ANN for Non-linear Data** – Decision system construction for linearly inseparable data. Hidden layer neuron activity analysis.
* **Project 3: Image Recognition (ANN)** – Multi-dimensional classification of monochrome images (character recognition). Evaluation using *Accuracy, Sensitivity (TPR), Specificity (TNR)*, and Confusion Matrix.
* **Project 4: Decision Trees** – Data classification system using decision trees, mapping decision boundaries and rule bases.
* **Project 5: Hopfield Network** – Autoassociative memory. Reconstructing and denoising distorted character images using Hebbian learning and pseudo-inverse rules.
